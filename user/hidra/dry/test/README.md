# Dry XDC Tests

This folder contains tests for `HidraDryXDCProducer`.

## Test files

- `HidraDryXDCProducerTest.cc`: basic behavior and parsing checks
- `HidraDryXDCProducerAdvancedTest.cc`: edge cases and robustness checks
- `HidraDryXDCProducerTestFixtures.hh`: shared generators/helpers

## Execution

- CTest filter: `HidraDryXDC`
- Executables:
	- `bin/tests/hidra_dry_xdc_producer_test`
	- `bin/tests/hidra_dry_xdc_producer_advanced_test`
